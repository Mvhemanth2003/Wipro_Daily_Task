Scenario
Student needed access to a specific skill details page.
Old Implementation
Manually parsed URLs using req.url.split('/').
New Implementation in Express
app.get('/skills/:id', (req, res) => {
    const skill = skillList.find(s => s.id === Number(req.params.id));
});
