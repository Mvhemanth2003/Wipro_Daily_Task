Transition into = the enter animation when a new route/component is mounted. In CSS: .page-enter → .page-enter-active.
Transition from = the exit animation when the current route/component is unmounted (navigating away). In CSS: .page-exit → .page-exit-active.
TransitionGroup + CSSTransition coordinate mounting/unmounting so both enter and exit animations run cleanly.
Using location and key={location.pathname} ensures unique transitions per route path.
